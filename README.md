# PromptWars Bot — AskAI ⚡

An AI-powered chatbot built with Gemini 1.5 Flash for the PromptWars Virtual Challenge.

## Features
- Real-time AI chat powered by Google Gemini
- Clean dark UI with typing indicators
- Conversation history support
- Mobile responsive

## Tech Stack
- Node.js + Express (Backend)
- Vanilla HTML/CSS/JS (Frontend)
- Google Gemini 1.5 Flash API

```bash
npm install
echo "GEMINI_API_KEY=your_key_here" > .env
